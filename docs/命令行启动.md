# 命令行启动

OpenCode Pocket Kit 支持注册一个独立命令，让你像使用 `opencode` 一样在任意终端中启动便携版 OpenCode，但不会占用 `opencode` 这个命令名。

## 默认命令名

默认命令名是：

```bat
opkcode
```

它来自根目录配置文件：

```text
command.conf
```

默认内容：

```ini
COMMAND_NAME=opkcode
```

如果你担心命令名冲突，可以先修改 `COMMAND_NAME`，例如：

```ini
COMMAND_NAME=opk-ai
```

然后重新运行：

```bat
注册命令.cmd
```

命令名限制：

- 必须以英文字母开头。
- 可以包含英文字母、数字、短横线和下划线。
- 不建议使用 `opencode`，避免与系统 OpenCode 命令冲突。

## 注册命令

双击根目录：

```bat
注册命令.cmd
```

它会完成两件事：

1. 在 `cmd\` 目录生成命令启动器，例如：

   ```text
   cmd\opkcode.cmd
   ```

2. 把 `cmd\` 目录加入当前 Windows 用户的 `Path` 环境变量。

注册完成后，请关闭已经打开的终端窗口，重新打开新的 PowerShell 或 CMD。Windows 已经打开的终端通常不会自动刷新用户 `Path`。

## 在当前目录启动 OpenCode

进入任意项目目录：

```bat
cd D:\Code\my-project
opkcode
```

此时 OpenCode 的工作目录就是当前终端所在目录：

```text
D:\Code\my-project
```

这和双击 `启动.cmd` 不同：

| 启动方式 | 工作目录 |
|---|---|
| 双击 `启动.cmd` | 包内 `projects\` 目录 |
| 终端执行 `opkcode` | 当前终端所在目录 |

## 工作目录的意义

OpenCode 和 agent 工具通常会把当前工作目录作为主要上下文。相对路径会以当前工作目录为基准。

例如在：

```bat
cd D:\Code\my-project
opkcode
```

之后，如果让 OpenCode 修改：

```text
src\main.ts
```

通常会指向：

```text
D:\Code\my-project\src\main.ts
```

因此建议每次先切换到真正的项目根目录，再运行 `opkcode`。

## 能否访问工作目录外的文件

OpenCode 进程继承当前 Windows 用户的文件权限。理论上，只要当前用户有权限，绝对路径指向的文件可以被访问。

不过代码代理通常以当前工作目录为主要上下文。为了避免路径误判，建议：

- 把工作目录设置为真正的项目根目录。
- 如果必须访问其他目录，提供完整绝对路径。
- 对涉及删除、覆盖、移动文件的操作保持谨慎。

## 与密钥库的关系

`opkcode` 和 `启动.cmd` 一样，都会调用便携启动脚本：

```text
scripts\20-启动.ps1
```

如果存在：

```text
vault\secrets.env.enc
```

启动时会要求输入密钥库密码，并把解密后的变量注入当前 OpenCode 进程。

不要直接运行：

```text
bin\opencode.exe
```

否则不会自动加载便携配置、密钥库和环境变量。

## 注销命令

如果不想继续使用命令行别名，双击：

```bat
注销命令.cmd
```

它会从当前用户的 `Path` 环境变量中移除本包的 `cmd\` 目录。

注销后，同样需要关闭并重新打开终端窗口。

## 常见问题

### 输入 opkcode 提示不是内部或外部命令

处理步骤：

1. 确认已经运行过 `注册命令.cmd`。
2. 关闭当前终端，重新打开一个新的终端。
3. 运行：

   ```bat
   echo %PATH%
   ```

   或在 PowerShell 中运行：

   ```powershell
   $env:Path
   ```

   检查是否包含本包的 `cmd\` 目录。

### 命令名和别的软件冲突

编辑：

```text
command.conf
```

改成新的命令名：

```ini
COMMAND_NAME=opk-ai
```

然后重新运行：

```bat
注册命令.cmd
```

### 移动便携包后命令失效

命令行别名目录写入的是当前包位置。移动整个便携包后，需要重新运行：

```bat
注册命令.cmd
```

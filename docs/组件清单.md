# 组件清单

本项目安装和管理的组件分为 OpenCode 本体、Oh-My-OpenAgent、TUI 插件、Comment Checker、Superpowers 和可选 Skills。

## OpenCode 本体

| 组件 | 包名 | 安装位置 | 说明 |
|---|---|---|---|
| OpenCode | `opencode-ai` | `npm-global\`、`bin\opencode.exe` | OpenCode 主程序 |

安装完成后，启动脚本优先使用：

```text
bin\opencode.exe
```

不要直接运行系统中的 `opencode`，否则可能不会加载便携目录中的配置和密钥。

## Oh-My-OpenAgent / OMO

| 组件 | 包名 | 安装位置 | 说明 |
|---|---|---|---|
| OMO Server Plugin | `oh-my-openagent` | `npm-global\`、`config\opencode\node_modules\` | OpenCode 插件和 agent 路由能力 |
| OMO TUI Plugin | `oh-my-openagent/tui` | `config\opencode\tui.json` | TUI 侧边栏和 TUI 命令能力 |

OpenCode 主配置位于：

```text
config\opencode\opencode.json
```

默认包含：

```json
{
  "$schema": "https://opencode.ai/config.json",
  "plugin": [
    "oh-my-openagent"
  ]
}
```

TUI 配置位于：

```text
config\opencode\tui.json
```

默认包含：

```json
{
  "plugin": [
    "oh-my-openagent/tui"
  ]
}
```

OMO 官方安装器会生成或维护：

```text
config\opencode\oh-my-openagent.json
```

该文件用于 OMO 的 agent、模型路由和提供商优先级。便携包不会手写生成空 OMO 路由文件。

## Comment Checker

| 组件 | 包名 | 安装位置 | 说明 |
|---|---|---|---|
| Comment Checker | `@code-yeongyu/comment-checker` | `npm-global\`、`config\opencode\node_modules\` | OMO 相关 hook 和检查能力 |

诊断脚本会检查以下位置：

```text
npm-global\comment-checker.cmd
npm-global\node_modules\.bin\comment-checker.cmd
config\opencode\node_modules\.bin\comment-checker.cmd
config\opencode\node_modules\@code-yeongyu\comment-checker\vendor\win32-x64\comment-checker.exe
```

## Superpowers

| 来源 | 缓存目录 | 安装内容 |
|---|---|---|
| `obra/superpowers` | `repo-cache\superpowers-official\` | Superpowers 插件与 Skills |

安装后常见位置：

```text
config\opencode\plugins\superpowers.js
config\opencode\skills\superpowers\
config\opencode\superpowers\
```

如果下载失败，可以检查 `logs\network-*.log`。

## 下载方式

可选组件下载脚本使用 Git sparse checkout，只拉取需要的路径。这样做是为了避免下载完整仓库。

下载缓存位于：

```text
repo-cache\
```

实际安装位置通常是：

```text
config\opencode\skills\
config\opencode\plugins\
```

## 安装失败时的判断

某个可选 Skill 下载失败，不代表 OpenCode 本体不可用。判断方式：

| 状态 | 影响 |
|---|---|
| `bin\opencode.exe` 存在 | OpenCode 本体可启动 |
| `opencode.json` 包含 `oh-my-openagent` | OMO server 插件已启用 |
| `tui.json` 包含 `oh-my-openagent/tui` | OMO TUI 插件已启用 |
| 某些 Skill 缺失 | 只影响对应 Skill，不影响基础对话 |
| Comment Checker 缺失 | 影响对应 hook，不影响 OpenCode 本体启动 |

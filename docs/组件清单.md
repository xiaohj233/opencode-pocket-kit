# 组件清单

本项目安装的组件分为 OpenCode 本体、Oh-My-OpenAgent、TUI 插件、Comment Checker 和可选 Skills。

## OpenCode 本体

| 组件 | 包名 | 安装位置 | 说明 |
|---|---|---|---|
| OpenCode | `opencode-ai` | `npm-global/`、`bin/opencode.exe` | OpenCode 主程序 |

安装完成后，`bin/opencode.exe` 是启动脚本使用的便携入口。

## Oh-My-OpenAgent / OMO

| 组件 | 包名 | 安装位置 | 说明 |
|---|---|---|---|
| OMO Server Plugin | `oh-my-openagent` | `npm-global/`、`config/opencode/node_modules/` | OpenCode 插件和 agent 路由能力 |
| OMO TUI Plugin | `oh-my-openagent/tui` | `config/opencode/tui.json` | TUI 侧边栏和 TUI 命令能力 |

OpenCode 主配置：

```json
{
  "plugin": [
    "oh-my-openagent"
  ]
}
```

TUI 配置：

```json
{
  "plugin": [
    "oh-my-openagent/tui"
  ]
}
```

OMO 官方安装器会生成：

```text
config\opencode\oh-my-openagent.json
```

该文件负责 OMO 的 agent、模型路由和提供商优先级。

## Comment Checker

| 组件 | 包名 | 安装位置 | 说明 |
|---|---|---|---|
| Comment Checker | `@code-yeongyu/comment-checker` | `npm-global/`、`config/opencode/node_modules/` | OMO 相关 hook 和检查能力 |

诊断脚本会检查以下位置：

```text
npm-global\comment-checker.cmd
config\opencode\node_modules\.bin\comment-checker.cmd
config\opencode\node_modules\@code-yeongyu\comment-checker\vendor\win32-x64\comment-checker.exe
```

## 下载方式

可选组件通过 `安装.cmd` 和 `更新.cmd` 自动下载。下载脚本会优先使用 Git sparse checkout，避免不必要地下载完整仓库。

下载缓存位于：

```text
repo-cache```

安装后的 Skills 位于：

```text
config\opencode\skills```
